# Maze Traversal Strategy Pattern Demo

## Requirements
- Java 21 (Amazon Corretto 21 or equivalent)
- Maven 3.6 or higher

## Quick Start

### Option 1: Run directly with Maven (recommended)
```bash
mvn clean compile exec:java
```

### Option 2: Build and run JAR
```bash
mvn clean package
java -jar target/maze-traversal-strategy-1.0-SNAPSHOT-jar-with-dependencies.jar
```
